Import Mice

Upload CSV or XLSX. Import runs in all-or-nothing mode. Required fields are marked *.

上传 CSV 或 XLSX。导入采用“全成或全失败（all-or-nothing）”模式。 必填字段用 * 标记。

One row = one mouse. Preferred format is per-locus columns (for example: Gpr82, Foxp3-Cre, Tet2, Lyz2-Cre) where each cell stores a genotype display (such as +/+, fl/fl, Cre/Y).

一行代表一只小鼠。推荐使用“每个 locus 一列”的格式（例如：Gpr82、Foxp3-Cre、 Tet2、Lyz2-Cre），每个单元格填写基因型展示值（如 +/+、fl/fl、Cre/Y）。

This import supports more loci per file. You can keep adding locus columns as needed (practically unlimited in normal usage).

本导入支持更多基因位点。你可以按需持续增加 locus 列（在常规使用中基本不限数量）。

If a locus column is present but the cell is empty, the locus is still added with a blank genotype (to fill in later). Strain-line template loci missing from the file are also created as empty placeholders.

若 Excel 中有 locus 列但单元格为空，仍会创建该位点行，基因型留空待填。 品系模板里未出现在表格中的位点，也会自动创建空占位行。

Legacy genotype_N_* slot columns are still supported for backward compatibility. After import, review genotype results in each Mouse detail page under Genotype Workspace.

旧版 genotype_N_* 槽位列仍可兼容导入。 导入后请在每只鼠的详情页 Genotype Workspace 中复核结果。
